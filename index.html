1. Как работает Web

Описание схемы:

  1. Клиент вводит запрос в браузере и нажимает Enter
  2. Браузер отправляет запрос пользователя в библиотеку DNS
  3. Браузер получает из DNS значение IP
  4. Браузер отправляет IP на сервер, на котором хранится страница
  5. Сервер отправляет браузеру файлы страницы (HTML, CSS и др.)
  6. Браузер рендерит (обрабатывает) полученные файлы и выводит их на экран пользователю

Класс

2. Синтаксис CSS

На первом этапе мы создаем большинство элементов в HTML, а далее стилизуем их с помощью CSS в 2 этапа:

  1. В CSS файле обращаемся к одному или нескольким HTML-элементам с помощью селектора/ов (по названию тега, значению атрибутов class или id, или вложенным селекторам, в которых есть и теги, и классы)
  2. Присваиваем данному селектору определенное свойство, которое обладает определенным значением (например, color: red)

Далее браузер считывает эти инструкции и преобразует их в визуальные формы на экране пользователя.

4. Селекторы
  1. Селекторы с названием тега в файле html
    • название селектора указывается БЕЗ точки в начале записи (например: body, p, div, span и т.д.)
    • далее внутри фигурных скобок {} указываются свойства и их значения

  2. Селекторы со значением атрибута class у определенного тега в файле html
    • название селектора указывается С точкой в начале записи (например: .card, .box, .container и т.д.)
    • далее внутри фигурных скобок {} указываются свойства и их значения
    • одно и то же значения атрибута class МОЖНО присваивать неограниченному числу html-тегов (это позволяет быстро менять стили большого кол-ва тегов одновременно)

  3. Селекторы со значением атрибута id у определенного тега в файле html
    • название селектора указывается С решеткой в начале записи (например: #intel, #ipsum, #termin и т.д.)
    • далее внутри фигурных скобок {} указываются свойства и их значения
    • одно и то же значения атрибута id НЕЛЬЗЯ присваивать неограниченному числу html-тегов

  4. Селекторы со значением атрибута class (либо id) и названием тега в файле html
    • примеры записи: .box p{}, .container div {}, #termin span{}

  5. Можно использовать комбинации всех предыдущих вариантов селекторов в рамках одной записи ЧЕРЕЗ ЗАПЯТУЮ:
    • #header, .card, .termin p {}

  6. Можно использовать селектор *{}, который обращается сразу ко все элементам веб-страницы

6. Работа с текстом

Шрифты делятся на:

  1. Встроенные
  2. Подключаемые (через Google fonts)
    • с помощью тега link, вкладываемого в тег head в HTML-файл
    • с помощью записи @import, добавляемой в CSS-файл

Свойства шрифтов:

  1. font-family — семейство шрифта
  2. font-size — размер шрифта (px, em, rem)
  3. font-weight — насыщенность шрифта
    • 100 - thin
    • 300 - lite
    • 400 - normal
    • 500 - medium
    • 600 - semibold
    • 700 - bold
    • 900 - black
  4. font-style — начертание
    • normal — обычное
    • italic — наклонное
    • bold — жирное

Свойства текста:

  1. text-decoration — подчеркивание
    • underline
    • overline
    • line-through
  2. line-height — высота строки
  3. letter-spacing — отступы между буквами
  4. word-spacing — отступы между словами
7. Цвета

За добавление цветов отвечает свойство color, которому присваиваются значения в форме:

  1. Слов: red, green, blue и т.д. Полный перечень цветов можно посмотреть на сайте

  2. Хэштега с rgb-кодом: #f8af00, #f023fjh и т.д.
    • первые 2 символа отвечают за красный цвет, вторые — за синий, третье — за зеленый
    • минимальное значение 00 — это черный цвет
    • максимальное значение ff — это красный/зеленый/голубой цвет

  3. Функции с численным rgb-кодом: rgb (117, 12, 255)
    • первое число отвечает за красный цвет, второе — за синий, третье — за зеленый
    • минимальное значение 00 — это черный цвет
    • максимальное значение 255 — это красный/зеленый/голубой цвет

  4. Функции с численным rgba-кодом: rgba (117, 12, 255, 1)
    • последнее значение варьируется в диапазоне 0..1 и отвечает за прозрачность
    • 0 — 100% прозрачность
    • 1 — 100% непрозрачность

8. Границы элементов

  1. Существуют следующие свойства для создания границ элементов:
    • border-width — ширина границы
    • border-style — начертание границы
      • solid — сплошное
      • dashed — пунктирное
      • dotted — точечное
    • Существует короткая запись, которая включает все три свойства:
      • border: 2px solid red
    • border-radius — радиус границы
    • Положение границы можно задавать отдельными свойствами:
      • border-bottom: left — граница слева
      • border-bottom: right — граница слева
      • border-bottom: bottom — граница слева
      • border-bottom: top — граница слева
  2. Все вышеперечисленные свойства можно комбинировать:
    • border-bottom-right-radius: 10px

9. Box Model

1. Концепция

Любому элементу соответствуют определенную параметры:

  1. ширина*высоту
  2. padding: 20px, 10px, 5px 0px — внутренний отступ элемента:
    • сверху — 20 пикселей
    • справа — 10 пикселей
    • снизу — 5 пикселей
    • слева — 0 пикселей
  3. border — граница элемента
  4. margin: 20px, 10px, 5px 0px — внешний отступ элемента:
    • сверху — 20 пикселей
    • справа — 10 пикселей
    • снизу — 5 пикселей
    • слева — 0 пикселей

Таким образом элемент с определенными параметрами ширины*высоту как бы "упакован" в коробку из трех слоев, отсюда и название box model.

☝️ Задав определенное значение width (или max-width) и задав margin: auto можно выравнять веб-страницу по центру на экране.

2. Свойство box-sizing

  1. content-box — значение, которое соответствует по умолчанию свойству box-sizing, из-за чего ширина width границы border прибавляется к значению ширины и высоты самого элемента
  2. border-box — значение, присвоив которое свойству box-sizing ширина width границы border отнимается от значения ширины и высоты самого элемента
10. Ссылки и псевдоселекторы

Стилизация ссылок

Стилизацию ссылок можно разделить на 2 вида:

  1. Статическая стилизация — задаем свойства ссылке как для обычного текста:
    • мы обращаемся к тегу a {}
    • и задаем свойства тексту ссылки (color, text-decoration и т.д.)
  2. Динамическая стилизация — добавляя к тегу или значению атрибута class псевдоселекторы, мы можем задать отображение ссылки (и другим элементам) в разных ситуациях:
    • :hover {} — меняем стиль ссылки при наведении курсора
    • :active {} — меняем стиль ссылки при нажатии на нее
    • :visited {} — меняем стиль ссылки после посещения прикрепленного к ней адреса
    • :focus {} — меняем стиль элемента (например, формы ввода текста) при нажатии на него

Стилизация кнопок

Аналогичным образом можно стилизовать кнопки, которые создаются с помоom. тега button и к которым можно прикреплять ссылки.

*Но есть некоторые особенности с отображением шрифтов и прозрачности

Добавление ссылок

У свойства display есть несколько значений:

  • block — превращает линейный элемент в блочный
  • inline — превращает линейный элемент в строчный
  • inline-block — линейный элементу сохраняет "линейность" (способность занимать ширину по кол-ву своего содержимого) но при этом к нему можно начать применять свойства блочных элементов, такие как margin и padding

Отдельно выделяется значение flex, которое позволяет располагать элементы в строку, менять их взаимное расположение и др.

13. Свойство Position

Свойство position устанавливает поведение элемента относительно окна браузера и относительно других элементов. Существуют следующие значения данного свойства:

  • static — используется по умолчанию, благодаря нему позиция элемента зависит от уровня его вложенности в структуре кода HTML
  • relative — позволяет менять положение элемента относительно САМОГО СЕБЯ с помощью дополнительных свойств:
    • top: 20px
    • bottom: 30px
    • left: 10px
    • right: 20px
  • absolute — перестает учитывать вложенность элемента, позволяет менять положение элемента относительно ОКНА БРАУЗЕРА или родительского элемента со значениями position: relative/absolute/fixed/sticky с помощью дополнительных свойств:
    • top: 20px
    • bottom: 30px
    • left: 10px
    • right: 20px
  • fixed — элемент начинает позиционироваться только относительно окна браузера и не меняет своего положения при пролистывании страницы
  • sticky — сочетание свойств fixed и absolute: при проматывании страницы элемент "прилипает" к верху окна браузера и не исчезает при проматывании
  • Адаптивная

14. Введение в адаптивную верстку с Media

Адаптивная верстка позволяет подгонять размеры элементов страницы в зависимости от размеров экрана устройства, с которого она открывается. Активируется с помощью команды @media, котороый мы присваиваем новые значения при выполнении определенных условий
